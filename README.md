# TranscriptIQ 

**YouTube Video Transcript Extractor** — Extract clean transcripts from YouTube videos in seconds.

## Features
- Paste any YouTube video link
- Extract full transcript instantly
- Simple and fast Streamlit interface
- Clean output for reading, copying, or analysis

## Tech Stack
- Python
- Streamlit
- youtube-transcript-api
- Pandas

## Setup

```bash
git clone https://github.com/Cashlin3/TranscriptIQ-Youtube-Video-Transcript-Extractor.git
cd TranscriptIQ-Youtube-Video-Transcript-Extractor
pip install -r requirements.txt
streamlit run app.py
```
